**Simon Game: A Memory Challenge**

A sleek, interactive web implementation of the classic Simon memory game. This project focuses on managing game logic, 
handling user-triggered events, and synchronizing visual/auditory feedback using jQuery.

Features
1. **Dynamic Sequence Generation**: Every round generates a new random color in the sequence, increasing difficulty as the level rises.
2. **Interactive UI**: Custom CSS animations provide visual feedback for button presses and game-over states.
3. **Audio Synchronization**: Integrated sound effects for each color to enhance user immersion.
4. **State Management**: Tracks user input against the generated pattern to validate progress in real-time.

Tech Stack
1. **HTML5 & CSS3**: Structured for a clean, centered gaming interface.
2. **JavaScript**: Core game logic, including sequence validation and level tracking.
3. **jQuery**: Used for DOM manipulation, event handling, and managing CSS transitions.

How to Play
1.  **Start**: Press any key on your keyboard to begin the game.
2.  **Observe**: Watch the sequence of colors and listen to the associated sounds.
3.  **Repeat**: Click the buttons in the same order as shown.
4.  **Advance**: If you match the sequence correctly, you move to the next level.
5.  **Game Over**: If you click the wrong color, the game ends, and you can press any key to restart.
